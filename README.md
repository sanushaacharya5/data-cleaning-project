# Data Cleaning & Preparation Project
This project focuses on cleaning and preparing raw data using Python, Pandas, and Jupyter Notebook.

# Tasks Performed
* Handled missing values
* Removed duplicate records
* Cleaned and organized dataset
* Exported cleaned dataset for analysis

# Tools & Technologies
* Python
* Pandas
* Jupyter Notebook
* Excel Dataset

# Files Included
* `raw_data.xlsx` → Original dataset
* `cleaned_data.xlsx` → Cleaned dataset
* `DataCleaningProject.ipynb` → Jupyter Notebook containing the code

## Learning Outcome
This project helped me understand the importance of data preprocessing and how clean data improves data analysis workflows.
